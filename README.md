# Firestationquiz backend service - Game-data  

## Goal of this service  

Provide data for the game with the API output.  
All questions needs to be generated randomly.  

## Outputformat

```json
{
  "option1": string,
  "option2": string,
  "option3": string,
  "option4": string,
  "ans": string,
  "imageUrl": string
}
```
